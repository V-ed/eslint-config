# eslint-config

Shared [ESLint config](https://eslint.org/docs/developer-guide/shareable-configs) for my projects.

## Install

```bash
npm install --save-dev @v-ed/eslint-config
```

## Usage

`.eslintrc.json`

```json
{
  "extends": ["@v-ed/eslint-config"], // or ["@v-ed"]
  "includes": {
    ...
  }
}
```
